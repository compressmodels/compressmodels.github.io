Problem

Classification models must run efficiently to be deployed in real-time embedded systems. Achieving low energy and low latency requirements typically requires multiple people-months of engineering effort.

Solution

moco is a mathematical optimization library that analyzes a model’s input data and derives rules that match the model’s predictions. Those rules replace model inference for portions of the input space. This results in avoided computation and produces lower energy and lower latency systems.

Use cases

  • Financial: Real-time credit card fraud detection; deciding whether to buy/sell/hold in high frequency trading systems.
  • Cybersecurity: specializing in preventing High-Volume Attacks and fast-propogating attacks (e.g DDOS, malware detection)
  • Energy: Predictive maintenance for smart grid sensors, battery charge/discharge prediction, fault detection.
  • Security (Cameras & Drones): Real-time threat detection in images/streams.
  • Defense: Edge threat detection (e.g., naval/submarine systems).
