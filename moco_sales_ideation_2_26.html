Problem

  • High-Volume Classification workloads on the revenue-critical path incur very large costs.
  • Autonomous Vehicles companies spend $100k / day running simulations.
  • An A100 GPU costs $15k, 100 cost $1.5M (one time).
  • It’s a scaling problem: some companies care about this because it means they can increase their throughput, some want to cut costs.

Product:

  • moco is a software package
  • input: a trained model, and a dataset
  • outputs: a faster & cheaper model. no accuracy loss, no retraining.
  • moco finds easy-to-classify data by scanning the intermediate activations and avoids full computation of the model in those cases.
  • 20-30% throughput/latency improvement on image & text classification tasks.
